Application

Building the application

Apps that get smarter over time

Smart application factories can rapidly scale up capabilities by decoupling the machine learning processes from application development.


Application developers can call endpoints provided by the data science function for predictions without knowing anything about machine learning or analytics.


In this model, the intelligence of the application is provided “as a service” through an API available to developers for predictions.


Data scientists, in turn, can use actual data from the live application to retrain and improve the inferences provided by their machine learning algorithms.


The applications ”get smarter” over time.
